# Smore UI/UX 디자인 참고 자료 분석

**기준**: 제공된 Smore 플랫폼 스크린샷 분석  
**목적**: Theme V2 업그레이드 시 디자인 가이드라인 및 컴포넌트 스펙 정의

## 🧩 컴포넌트 스펙

### 1. 입력 필드 (Input Fields)

#### 공통 스타일
- **배경**: 흰색 (`#FFFFFF`)
- **테두리**: 둥근 모서리 (`border-radius: 8px ~ 12px`)
- **아이콘**: 왼쪽 정렬, 보라색/파란색
- **플레이스홀더**: 회색 텍스트
- **레이블**: 오른쪽 정렬, 진한 회색

#### 타입별 스펙

**텍스트 입력**
- 아이콘: 연필/펜
- 플레이스홀더: "ABC"
- 높이: `44px` (터치 타깃)

**이메일 입력**
- 아이콘: 봉투
- 플레이스홀더: "a@b.com"
- 유효성 검사: 실시간

**전화번호 입력**
- 아이콘: 전화기
- 플레이스홀더: "1234-5678"
- 포맷팅: 자동 하이픈

**날짜 입력**
- 아이콘: 달력
- 플레이스홀더: "2025-11-07"
- 피커: 모달 또는 인라인

**시간 입력**
- 아이콘: 시계
- 플레이스홀더: "17:06"
- 피커: 12시간제, 10분 단위

**숫자 입력**
- 아이콘: `#`
- 플레이스홀더: "123456"
- 키보드: 숫자 키패드

**드롭다운**
- 아이콘: 없음
- 화살표: 오른쪽 하단 (▼)
- 높이: `44px`

**별점**
- 아이콘: 별 5개 (기본)
- 색상: 보라색 채움/테두리
- 크기: `24px` ~ `32px`

**척도 (Scale)**
- 최소값: 왼쪽 표시
- 최대값: 오른쪽 표시
- 슬라이더: 원형 썸
- 색상: 보라색

---

### 2. 선택 필드 (Choice Fields)

#### 단일 선택 (Single Choice)
- **타입**: 라디오 버튼 또는 체크박스 (단일 선택)
- **스타일**: 흰색 카드, 보라색 테두리
- **텍스트**: "ABC" (보라색)
- **선택 상태**: 보라색 배경 또는 체크 표시

#### 다중 선택 (Multiple Choice)
- **타입**: 체크박스
- **스타일**: 단일 선택과 동일
- **선택 상태**: 여러 개 선택 가능

#### 이미지 선택 (Image Select)
- **레이아웃**: 2열 그리드
- **이미지**: 썸네일, 둥근 모서리
- **선택 표시**: 보라색 테두리 또는 오버레이

---

### 3. 버튼 (Buttons)

#### 주요 버튼 (Primary)
- **배경**: 보라색 (`#6B46C1`)
- **텍스트**: 흰색
- **모서리**: 둥근 모서리 (`border-radius: 12px`)
- **높이**: `48px` (터치 타깃)
- **아이콘**: 오른쪽 화살표 (`>`)
- **예시**: "> 설문 시작하기", "> 다음"

#### 보조 버튼 (Secondary)
- **배경**: 흰색
- **테두리**: 보라색
- **텍스트**: 보라색

#### 텍스트 버튼
- **배경**: 투명
- **텍스트**: 보라색 또는 파란색
- **밑줄**: 호버 시

---

### 4. 카드 (Cards)

#### 질문 카드
- **배경**: 흰색
- **패딩**: `24px`
- **그림자**: `0 2px 8px rgba(0,0,0,0.1)`
- **모서리**: `12px`

#### 통계 카드
- **배경**: 색상별 (보라, 파랑, 주황, 초록, 빨강)
- **아이콘**: 왼쪽 상단
- **숫자**: 큰 폰트 (`24px` ~ `32px`)
- **레이블**: 작은 폰트 (`14px`)

---

### 5. 네비게이션 (Navigation)

#### 상단 탭
- **활성 탭**: 보라색 밑줄
- **비활성 탭**: 회색 텍스트
- **예시**: "스타일", "커버", "문제", "엔딩", "퍼블리싱"

#### 사이드바
- **너비**: `240px` (관리자)
- **배경**: 연한 회색
- **활성 항목**: 보라색 배경
- **아이콘**: 왼쪽 정렬

#### 진행률 바
- **배경**: 연한 회색
- **진행**: 보라색/파란색
- **텍스트**: "3/5" (현재/전체)
- **높이**: `4px` ~ `8px`

---

### 6. 모달/다이얼로그

#### 질문 편집 모달
- **배경**: 반투명 오버레이
- **카드**: 중앙 정렬, 흰색
- **너비**: `600px` (데스크톱), `100%` (모바일)
- **닫기**: X 버튼 또는 배경 클릭

---

### 7. 차트 (Charts)

#### 파이 차트
- **색상**: 보라색, 파란색 계열
- **범례**: 오른쪽 또는 하단
- **라벨**: 각 섹션에 퍼센트 표시

#### 막대 차트
- **색상**: 보라색 그라데이션
- **축**: X축 (카테고리), Y축 (값)

## 🎯 인터랙션 패턴

### 호버 효과
- **버튼**: 배경색 약간 진하게 또는 그림자 증가
- **카드**: 약간 올라오는 효과 (`transform: translateY(-2px)`)
- **링크**: 밑줄 표시

### 포커스 상태
- **입력 필드**: 보라색 테두리 (`2px solid #6B46C1`)
- **버튼**: 외곽선 (`outline: 2px solid #6B46C1`)

### 로딩 상태
- **스피너**: 보라색 원형 애니메이션
- **스켈레톤**: 회색 플레이스홀더

### 에러 상태
- **메시지**: 빨간색 배경, 흰색 텍스트
- **입력 필드**: 빨간색 테두리
- **예시**: "필수 항목입니다."

## 🔄 애니메이션

### 전환 효과
- **기본**: `200ms ease-in-out`
- **빠른 전환**: `150ms ease-in-out`
- **느린 전환**: `300ms ease-in-out`

### 페이드 인
- **순차적 등장**: 각 요소마다 `100ms` 지연
- **사용 위치**: 질문 목록, 통계 카드

## 📝 참고 사항

1. **색상 일관성**: 모든 인터랙티브 요소는 보라색 계열 사용
2. **터치 타깃**: 최소 `44px` 높이 유지
3. **모바일 퍼스트**: 모든 컴포넌트는 모바일 기준으로 설계
4. **접근성**: 색상 대비 비율 AA 준수 (4.5:1 이상)
5. **애니메이션**: 부드럽고 빠른 전환 효과
