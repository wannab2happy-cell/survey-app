## 🚨 문제 발견 시 대응

### 문제 발견 시 즉시 확인
1. **기능이 사라진 경우**
   - `FEATURE_FREEZE.md`에서 해당 기능 확인
   - 최근 변경 사항 확인
   - 즉시 복원

2. **버튼이 작동하지 않는 경우**
   - `onClick` 핸들러 확인
   - `disabled` 속성 확인
   - 조건부 렌더링 확인

3. **페이지가 표시되지 않는 경우**
   - 라우트 확인
   - 컴포넌트 import 확인
   - 조건부 렌더링 확인

## 📝 체크리스트 사용 방법

1. **수정 전**: 이 체크리스트의 "수정 전 체크리스트"를 모두 확인
2. **수정 중**: 이 체크리스트의 "수정 중 체크리스트"를 참고하며 수정
3. **수정 후**: 이 체크리스트의 "수정 후 체크리스트"를 모두 확인

**⚠️ 중요**: 모든 체크리스트를 완료한 후에만 수정을 완료한 것으로 간주합니다.
