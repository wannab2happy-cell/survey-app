## 🛠️ 설문 빌더 (Survey Builder)

### 1. 탭 네비게이션 (SurveyBuilder.jsx)
**파일**: `client/src/components/SurveyBuilder.jsx`

| 순서 | 탭 ID | 탭 라벨 | 상태 |
|------|-------|---------|------|
| 1 | `style` | 스타일 | ✅ 필수 |
| 2 | `cover` | 커버 | ✅ 필수 |
| 3 | `questions` | 문제 | ✅ 필수 |
| 4 | `ending` | 엔딩 | ✅ 필수 |
| 5 | `publishing` | 퍼블리싱 | ✅ 필수 |

**⚠️ 주의**: 이 탭들은 절대 삭제하거나 순서를 변경하면 안 됩니다.

### 2. 퍼블리싱 탭 기능 (publishing)

#### 2.1 설문 진행 상태
- 비활성화 버튼 (inactive)
- 바로 진행 버튼 (active)
- 예약 진행 버튼 (scheduled)
- 일시 정지 버튼 (paused)

#### 2.2 링크 공유 및 QR 코드
- 설문 URL 입력 필드
- 복사 버튼
- QR 코드 표시 영역

#### 2.3 공개 설정
- 공개 설문 토글 (isPublic)
- 비밀번호 보호 토글 (password)
- 비밀번호 입력 필드

#### 2.4 저장 및 공유
- 설문 저장 버튼
- 미리보기 버튼
- 공유 버튼

**⚠️ 주의**: 이 기능들은 절대 삭제하면 안 됩니다.

### 3. 커버 탭 기능 (cover)
- 로고 업로드
- 설문지 제목 입력
- 부제목 입력
- 타이틀 이미지 업로드
- 설문 참여 수 표시 토글
- 설문 시작하기 버튼 프리뷰

### 4. 문제 탭 기능 (questions)
- 질문 추가 버튼 (항상 표시)
- 질문 유형 선택 모달
- 질문 목록 (QuestionCard)
- 질문 이미지 업로드
- 옵션 이미지 업로드
- 개인정보 수집 섹션

### 5. 엔딩 탭 기능 (ending)
- 완료 제목 입력
- 완료 설명 문구 입력
- 완료 페이지 이미지 업로드
- 외부 사이트 연결 (linkUrl, linkText)

### 6. 스타일 탭 기능 (style)
- 브랜딩 및 디자인 설정
- 색상 설정 (primary, secondary, tertiary)
- 폰트 스타일
- 버튼 모양
- 로고 이미지
- 배경 이미지
- Head/Foot 커스터마이징
