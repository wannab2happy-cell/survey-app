# 설문 플랫폼 디자인 시스템 재구성 제안서

## 🎯 목표
일관되고 깔끔한 디자인 시스템을 구축하여 사용자 경험을 향상시킵니다.

---

## 📋 현재 상황 분석

### 문제점
- 디자인이 여러 곳에서 엉켜있음
- 일관성 없는 색상/간격/타이포그래피 사용
- 컴포넌트 간 스타일 불일치

### 기존 리소스
- ✅ `DESIGN_REFERENCE.md` - Smore 기반 디자인 가이드
- ✅ `theme.css` - CSS 변수 정의
- ✅ Tailwind CSS 설정

## 🎨 디자인 시스템 제안

### 1. 색상 시스템

#### Primary Colors (주요 색상)
```
Primary:     #6B46C1 (보라색) - 메인 액션, 강조
Secondary:   #8B5CF6 (연한 보라) - 보조 액션
Accent:      #3B82F6 (파란색) - 정보, 링크
```

#### Neutral Colors (중립 색상)
```
Text Primary:   #111827 (진한 회색) - 제목, 본문
Text Secondary: #6B7280 (중간 회색) - 부제목, 설명
Text Tertiary: #9CA3AF (연한 회색) - 플레이스홀더
Border:        #E5E7EB (연한 회색) - 테두리
Background:    #F9FAFB (아주 연한 회색) - 배경
White:         #FFFFFF (흰색) - 카드, 입력 필드
```

#### Status Colors (상태 색상)
```
Success: #10B981 (초록) - 성공 메시지
Warning: #F59E0B (주황) - 경고 메시지
Error:   #EF4444 (빨강) - 에러 메시지
Info:    #3B82F6 (파랑) - 정보 메시지
```

### 2. 타이포그래피

#### 폰트 크기
```
H1: 32px (2rem) - 페이지 제목
H2: 24px (1.5rem) - 섹션 제목
H3: 20px (1.25rem) - 서브 섹션 제목
Body: 16px (1rem) - 본문
Small: 14px (0.875rem) - 설명, 레이블
Tiny: 12px (0.75rem) - 작은 텍스트
```

#### 폰트 굵기
```
Bold: 700 - 제목
Semibold: 600 - 부제목, 강조
Medium: 500 - 레이블
Regular: 400 - 본문
```

### 3. 간격 시스템 (Spacing)

```
xs:  4px  (0.25rem)
sm:  8px  (0.5rem)
md:  12px (0.75rem)
lg:  16px (1rem)
xl:  24px (1.5rem)
2xl: 32px (2rem)
3xl: 48px (3rem)
```

### 4. 둥근 모서리 (Border Radius)

```
sm:  4px  - 작은 요소
md:  8px  - 기본 버튼, 입력 필드
lg:  12px - 카드
xl:  16px - 큰 카드
full: 9999px - 원형 버튼
```

### 5. 그림자 (Shadows)

```
sm:  0 1px 2px rgba(0,0,0,0.05)   - 작은 요소
md:  0 2px 8px rgba(0,0,0,0.1)    - 카드
lg:  0 4px 16px rgba(0,0,0,0.15)  - 모달
xl:  0 8px 24px rgba(0,0,0,0.2)   - 큰 모달
```

## 🧩 컴포넌트 디자인 가이드

### 1. 버튼 (Buttons)

#### Primary Button
- 배경: Primary 색상 (#6B46C1)
- 텍스트: 흰색
- 패딩: 12px 24px
- 둥근 모서리: 8px
- 호버: 배경색 약간 진하게

#### Secondary Button
- 배경: 흰색
- 테두리: Primary 색상 2px
- 텍스트: Primary 색상
- 패딩: 12px 24px

#### Text Button
- 배경: 투명
- 텍스트: Primary 색상
- 호버: 밑줄

### 2. 입력 필드 (Input Fields)

- 배경: 흰색
- 테두리: 2px solid #E5E7EB
- 둥근 모서리: 8px
- 패딩: 12px 16px
- 높이: 44px (터치 타깃)
- 포커스: 테두리 Primary 색상, 그림자

### 3. 카드 (Cards)

- 배경: 흰색
- 패딩: 24px
- 둥근 모서리: 12px
- 그림자: md
- 호버: 약간 올라오는 효과 (translateY(-2px))

### 4. 모달 (Modals)

- 배경 오버레이: rgba(0,0,0,0.5)
- 카드: 중앙 정렬, 흰색
- 너비: 600px (데스크톱), 100% (모바일)
- 패딩: 24px
- 둥근 모서리: 16px

## 💬 협의 사항

### 결정이 필요한 항목

1. **색상 팔레트**
   - 현재 Primary 색상 (#6B46C1) 유지할까요?
   - 다른 색상 선호하시나요?

2. **버튼 스타일**
   - 둥근 모서리 정도 (8px vs 12px)?
   - Pill 형태 (완전히 둥근) 선호하시나요?

3. **카드 스타일**
   - 그림자 강도?
   - 호버 효과 필요할까요?

4. **타이포그래피**
   - 폰트 패밀리 선호하시나요? (Pretendard, Noto Sans KR 등)
   - 폰트 크기 적절한가요?

5. **우선순위**
   - 어떤 페이지부터 시작할까요?
   - 참가자 페이지 우선? 관리자 페이지 우선?
